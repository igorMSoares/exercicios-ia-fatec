# Inteligência Artificial - FATEC

## Trabalho 3 - Aproximação Funcional com Backpropagation

Uso de backpropagation em rede neural para aproximação funcional.

## Setup

### Binder

Clique [aqui](https://mybinder.org/v2/gh/igorMSoares/exercicios-ia-fatec/HEAD?urlpath=%2Fdoc%2Ftree%2Ftrabalho3%2Ffatec-ia-trabalho3-aproximacao-funcional.ipynb) para executar o código interativamente na nuvem através do serviço [Binder](mybinder.org)

### Jupyter Notebook

- Instalar [Jupyter Notebook](https://jupyter.org/install)
```bash
pip install notebook
```
- Iniciar Notebook
```bash
jupyter notebook
```

- Abrir no Notebook o arquivo `./fatec-ia-trabalho3-aproximacao-funcional.ipynb`
- Executar cada célula do arquivo
