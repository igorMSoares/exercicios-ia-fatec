# Inteligência Artificial - FATEC

## Trabalho 4 - Reconhecimento de caracteres utilizando Backpropagation

Uso de backpropagation em rede neural para reconhecimento de caracteres.

## Setup

### Binder

Clique [aqui](https://mybinder.org/v2/gh/igorMSoares/exercicios-ia-fatec/HEAD?urlpath=%2Fdoc%2Ftree%2Ftrabalho4%2Ffatec-ia-trabalho4-reconhecimento-caracteres.ipynb) para executar o código interativamente na nuvem através do serviço [Binder](mybinder.org)

### Jupyter Notebook

- Instalar [Jupyter Notebook](https://jupyter.org/install)
```bash
pip install notebook
```
- Iniciar Notebook
```bash
jupyter notebook
```

- Abrir no Notebook o arquivo `./fatec-ia-trabalho4-reconhecimento-caracteres.ipynb`
- Executar cada célula do arquivo
